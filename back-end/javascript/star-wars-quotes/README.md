# star-wars-quotes
My first node api

Dependencies
-------------- 
- npm install express --save
- npm install nodemon --save-dev
- npm install body-parser --save
- npm install mongodb --save
- npm install ejs --save


nodemon auto build
------------------

In "package.json"

{
  //...
  "scripts": {
    "dev": "nodemon server.js"
  }
  // ...
}

build
----------------
run npm install

run
----------------
npm run dev
